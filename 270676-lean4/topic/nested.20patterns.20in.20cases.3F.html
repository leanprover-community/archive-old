Leonardo de Moura (Mar 08 2021 at 14:37):

Lean 4 has a match tactic. The syntax is similar to match expressions

def f : Nat â†’ Nat
  | 0   => 1
  | 1   => 2
  | n+2 => 3

theorem ex (n : Nat) : f n > 0 := by
  match n with
  | 0  => simp [f]
  | 1  => simp [f]
  | Nat.succ (Nat.succ _) => simp [f]
