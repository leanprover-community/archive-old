Maybe there's a more straightforward way but this works:

def myBar : Bar where
  bar := loop where
         loop | 0 => Nat.le_refl 0
              | n + 1 => Nat.le_trans (loop n) n.le_succ

(Alternatively, let rec instead of where.)
