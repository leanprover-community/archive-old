Topic: ✔ tactic to name variables with daggers

{% raw %}

Siddhartha Gadgil (Nov 03 2022 at 14:29):

Is there a tactic in Lean to assign names to the variables automatically generated by intro, cases, induction etc. Essentially with the effect of 

let n := n✝
have eql : n✝ = n := by rfl
rw [eql]

It seems that this can be implemented using getUnusedVar, MVarId.define (for the let) and rewrite, but I wanted to check if it already exists or there is an easier way to implement this.

Mario Carneiro (Nov 03 2022 at 14:29):

rename_i x1 ... xn will rename the last n inaccessible variables as x1 ... xn

Mario Carneiro (Nov 03 2022 at 14:30):

also case _ x1 ... xn => or next x1 ... xn => can be used after a tactic that introduces inaccessible variables
