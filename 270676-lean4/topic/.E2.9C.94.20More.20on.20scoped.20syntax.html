Leonardo de Moura (Jun 29 2022 at 18:38):

This is expected. open Foo opens the namespace Foo and activates scoped attributes in Foo.
open scoped Foo just activates the scoped attributes. Example:

namespace Foo

def foo : Nat → Nat := id
def A : Nat := 10
scoped prefix:20 " `` " => foo

end Foo

open scoped Foo
#check A -- Error
open Foo
#check A -- Ok
