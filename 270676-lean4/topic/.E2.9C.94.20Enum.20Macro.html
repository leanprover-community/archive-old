You can use sepBy() (no optional trailing comma):

syntax "enum" ident "{" sepBy(ident, ",") "}" ";": command

macro_rules
  | `(enum $name:ident { $[$ids:ident],* };) =>
    `(inductive $name where $[| $ids:ident]*)

enum E { X, Y, Z };
