Eric Wieser (Feb 03 2021 at 15:02):

A similar example is:

example (a: ℕ) (b: ℕ) (s: ℕ) (h : b ≤ a ^ s):
  ↑(a ^ s - b) = (a: ℚ) ^ s - (b: ℚ) :=
begin
  rw nat.cast_sub h,
  simp,
end

Eric Wieser (Feb 03 2021 at 15:02):

Kevin Buzzard (Feb 03 2021 at 17:45):

import tactic
import data.rat.basic

example (a: ℕ) (b: ℕ) (s: ℕ) (h: 0 < a ^ s - b):
  ↑(a ^ s - b) = (a: ℚ) ^ s - (b: ℚ) :=
begin
  rw nat.cast_sub,
  { norm_cast },
  { exact le_of_lt (nat.lt_of_sub_pos h) }
end
