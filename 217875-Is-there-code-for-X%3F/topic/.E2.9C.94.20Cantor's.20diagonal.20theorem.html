Kyle Miller (Apr 29 2022 at 00:33):

(There's a small technicality, though, since (A B : Type*) means (A : Type*) (B : Type*), so both A and B can be types in different type universes.)

Kyle Miller (Apr 29 2022 at 00:42):

I didn't really realize that placeholders are filled per argument in general:

structure foo (α : Type) :=
(val : α)

def foo.get_nat (x : foo ℕ) : ℕ := x.val
def foo.get_list (x : foo (list ℕ)) : list ℕ := x.val

def test (x y : foo _) (h : x.get_nat = y.get_list.length) : true := trivial
-- the type of test is ∀ (x : foo ℕ) (y : foo (list ℕ)), x.get_nat = y.get_list.length → true
