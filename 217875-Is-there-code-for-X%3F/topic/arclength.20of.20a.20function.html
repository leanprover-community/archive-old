Jireh Loreaux (Apr 19 2022 at 18:54):

This involves considerably more machinery, but I think the API is developed relatively well by this point. Here, you perhaps want to work with a function f : ℝ → ℝ × ℝ which is continuously differentiable (i.e., (h : cont_diff ℝ 1 f). And then your integrand is ∥deriv f∥. You could take this (the interval_integral of this norm) as a definition of arc length if you wanted (although it's not going to be well-behaved / defined in more general circumstances).
