Anatole Dedecker (Sep 12 2022 at 14:16):

Here is a cleaner proof with the same assumptions

import topology.continuous_on
import topology.separation

open_locale topological_space

lemma is_open_set_of_eventually_nhds_within' [Œ± : Type*] [topological_space Œ±] [t1_space Œ±]
  {p : Œ± ‚Üí Prop} : is_open {x | ‚àÄ·∂† y in ùìù[‚â†] x, p y} :=
begin
  rw is_open_iff_mem_nhds,
  intros a ha,
  filter_upwards [eventually_nhds_nhds_within.mpr ha] with b hb,
  rcases eq_or_ne a b with (h | h),
  { subst h, exact ha },
  { rw h.symm.nhds_within_compl_singleton at hb,
    exact eventually_nhds_within_of_eventually_nhds hb }
end

Junyan Xu (Sep 12 2022 at 17:57):

A T0 counterexample is the affine line: an infinite space with a distinguished "generic" point whose nonempty open sets are cofinite sets containing the generic point. If you take p to be true at all points except at the generic point, then {x | ‚àÄ·∂† y in ùìù[‚â†] x, p y} is the singleton {generic point}, which is not open.
