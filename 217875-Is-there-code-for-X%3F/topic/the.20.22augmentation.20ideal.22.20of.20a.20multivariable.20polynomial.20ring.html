Kevin Buzzard (Aug 06 2022 at 13:31):

If RRR is a commutative ring and SSS is a type, then the multivariable polynomial ring R[S]:=R[S]:=R[S]:= mv_polynomial S R of polynomials in variables indexed by SSS has an ideal generated by all the variables. This ideal is the kernel of the ring homomorphism sending a multivariable polynomial to its constant coefficient, and the latter morphism is a surjective ring morphism from R[S]R[S]R[S] to RRR. Do we have this ideal, and a name for this ideal? This came up as a possible route to removing this sorry.
