David Wärn (Jul 04 2020 at 19:09):

This is a job for control theory

def foo {α β γ} (f : α → β → γ) (as : list α) (bs : list β) : list γ := f <$> as <*> bs
#reduce foo (+) [0, 1, 2] [10, 20, 30] -- [10, 20, 30, 11, 21, 31, 12, 22, 32]
