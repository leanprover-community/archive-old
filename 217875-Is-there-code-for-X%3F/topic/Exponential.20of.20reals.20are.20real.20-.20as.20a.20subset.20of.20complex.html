Heather Macbeth (Jul 07 2022 at 02:42):

docs#complex.exp_of_real_im is this but with the hypothesis in a different format:

lemma expOfRealIsReal (b:ℂ) (bIsReal:b.im=0) : (exp b).im=0 :=
begin
  obtain ⟨x, y⟩ := b,
  have : y = 0 := bIsReal,
  rw this,
  exact exp_of_real_im x,
end
