eq_of_forall_dist_le
  (λ ε hε, (dist_zero_right _).trans_le $ (real.norm_eq_abs _).trans_le (h ε hε).le)

Patrick Johnson (Jul 28 2022 at 16:29):

|x| ≤ ε is a more general version of it.

import topology.metric_space.basic

example {x : ℝ} (h : ∀ (ε > 0), |x| ≤ ε) : x = 0 :=
eq_of_forall_dist_le (by rwa real.dist_0_eq_abs)
