Heather Macbeth (Oct 11 2021 at 18:43):

@The Leanest Do you want something like this?

import analysis.inner_product_space.pi_L2

def v : euclidean_space ℂ (fin 3) := ![(1:ℂ), 2, 3]
def w : euclidean_space ℂ (fin 3) := ![(2:ℂ), 1, 0]

example : ⟪v, w⟫_ℂ = 4 :=
by norm_num [v, w, fin.sum_univ_succ]
