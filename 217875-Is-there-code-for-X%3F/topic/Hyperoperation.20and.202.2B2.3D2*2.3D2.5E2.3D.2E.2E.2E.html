Looking closer, it should instead be

def hyperoperation : ℕ → ℕ → ℕ → ℕ
| 0 _ b := 1 + b
| 1 a 0 := a

| (n + 1) a (b + 1) := hyperoperation n a (hyperoperation (n + 1) a b)
