If you don't need the countability, we probably already know that the convex hull of any SSS is the intersection of all the half-spaces containing SSS. Now apply this to S={ (x,y)∣y≥f(x) }S = \{\,(x, y) \mid y \ge f(x)\,\}S={(x,y)∣y≥f(x)}. It's pretty easy to see that any half-space containing SSS has to be of the form { y≥a(x) }\{\,y \ge a(x)\,\}{y≥a(x)} for some affine aaa, and then that SSS is this intersection means exactly that f(x)f(x)f(x) is the supremum of all the a(x)a(x)a(x), I think.
