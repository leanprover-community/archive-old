Adam Topaz (Feb 01 2022 at 03:54):

If you want a less automatic proof:

example : A ×ˢ (B ∩ C) = (A ×ˢ B) ∩ (A ×ˢ C) :=
begin
  ext,
  split,
  { rintros ⟨h1,h2,h3⟩,
    exact ⟨⟨h1,h2⟩,⟨h1,h3⟩⟩ },
  { rintros ⟨⟨h1,h1'⟩,⟨h2,h2'⟩⟩,
    exact ⟨h1,h1',h2'⟩ }
end
