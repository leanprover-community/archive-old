Yury G. Kudryashov (Feb 11 2022 at 23:37):

Here is the statement:

lemma cauchy_seq.comp_injective {α : Type*} [uniform_space α] {u : ℕ → α} (hu : cauchy_seq u)
  {f : ℕ → ℕ} (hf : function.injective f) : cauchy_seq (u ∘ f)

Yury G. Kudryashov (Feb 11 2022 at 23:38):

And the proof is

open function filter

lemma function.injective.tendsto_at_top {f : ℕ → ℕ} (hf : injective f) :
  tendsto f at_top at_top :=
by simpa only [← nat.cofinite_eq_at_top] using hf.tendsto_cofinite

lemma cauchy_seq.comp_injective {α : Type*} [uniform_space α] {u : ℕ → α} (hu : cauchy_seq u)
  {f : ℕ → ℕ} (hf : injective f) : cauchy_seq (u ∘ f) :=
hu.comp_tendsto hf.tendsto_at_top
