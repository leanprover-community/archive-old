Yury G. Kudryashov (Feb 06 2022 at 02:28):

A natural way to state many inequalities is: a ≤ b and a = b only if P. E.g., distance is nonnegative and equals zero only if the points are equal.
Then we use simple inequalities to prove another one by transitivity: a ≤ b ≤ c. In a paper, after a proof like this one can say "so, if a = c, then we have a = b = c, thus P and Q".

Yury G. Kudryashov (Feb 06 2022 at 02:29):

How can we translate this into Lean?

Yury G. Kudryashov (Feb 06 2022 at 02:30):

Something like this?

def le_with_eq [has_le α] (a b : α) (p : Prop) : a ≤ b ∧ (a = b → p)

Yury G. Kudryashov (Feb 06 2022 at 02:31):

Then we can have le_with_eq.le to get the inequality.

Yury G. Kudryashov (Feb 06 2022 at 02:31):

And le_with_eq.trans/le_with_eq.trans_le/has_le.le.trans_le_with_eq

Floris van Doorn (Feb 07 2022 at 09:20):

(1) I'm not sure whether this is worth a new definition, it might be easier to just state this as two separate lemmas. But I don't mind having it...
(2) le_with_eq.trans_le p q = p.le.trans q?
(3) I think in many cases p is equivalent with a = b
(4) You probably want multiple versions of trans, like le_with_eq.trans_same: le_with_eq a b p -> le_with_eq b c p -> le_with_eq a c p
