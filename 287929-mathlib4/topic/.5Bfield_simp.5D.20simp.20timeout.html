David Renshaw (Jan 21 2023 at 16:17):

While working on porting field_simp, I noticed the following behavior:

import Mathlib.Algebra.Ring.Units

-- If you comment out any of the below imports, then the proof below succeeds.
import Mathlib.Data.Rat.Basic
import Mathlib.Order.Bounds.OrderIso
import Mathlib.Algebra.Field.Basic
import Mathlib.Algebra.Order.Field.Defs

variable {R : Type _} [CommRing R] (a b : R) (u₁ u₂ : Rˣ)

-- Proof succeeds only if at least one of the above imports is commented out.
-- Otherwise we get the error:
-- timeout at 'whnf', maximum number of heartbeats (200000) has been reached
--  (use 'set_option maxHeartbeats <num>' to set the limit)
example : a /ₚ u₁ = b /ₚ u₂ ↔ a * u₂ = b * u₁ :=
by simp [divp_eq_iff_mul_eq, divp_mul_eq_mul_divp]

Lean 3 has no trouble at all with this example.

Anyone have any ideas about what's going wrong?
