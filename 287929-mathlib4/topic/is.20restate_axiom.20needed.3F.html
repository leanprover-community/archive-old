Mario Carneiro (Mar 09 2022 at 23:27):

I think it is still useful, because referencing these expressions leads to messy types:

structure A :=
  x : Nat
  a' : x = 1 := by trivial

example (z : A) : z.x = 1 := by
  have := z.a'
  -- z : A
  -- this : autoParam (A.x z = 1) _auto✝
  -- ⊢ A.x z = 1
  exact this
