Kevin Buzzard (Jan 18 2023 at 12:11):

Eric Wieser said:

So you'd be against this suggestion then?

This has already come up many times in the porting process and what we've been doing up until now is removing @[simp] from the definition, and then writing the equation lemmas manually and tagging them with @[simp] instead.

import Mathlib.Tactic.Conv

def foo (a : Nat) : Nat :=  2 * a

@[simp] theorem foo_def (a : Nat) : foo a = 2 * a := rfl

#simp foo -- does not unfold
#simp foo 1 -- does

def bar {α : Type _} (a : Nat → α) (i : Nat) : α :=
a (i * 2)

@[simp] theorem bar_def {α : Type _} (a : Nat → α) (i : Nat) :
    bar a i = a (i * 2) := rfl

#simp bar -- does not unfold
#simp @bar -- does not unfold
