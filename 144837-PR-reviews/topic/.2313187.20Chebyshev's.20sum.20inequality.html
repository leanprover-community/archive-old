The idea is that it means that f and g vary together. Note that this is a symmetric relation.

Yaël Dillies (Jan 17 2023 at 15:17):

If f : ι → α, g : ι → β, it means that one can order ι such that both f and g are monotone. The point is that

  • actually one cannot order ι like so in full generality
  • this condition doesn't need an order on ι at all to be stated and proved
