I think there are additional issues with search being raised here, but to answer this one: you can use the CTRL/CMD + . shortcut to narrow to the current compose box recipient. So if the topic auto-complete suggests a topic you think might be relevant, you can choose it from the auto-complete and then use the shortcut to navigate to it.
