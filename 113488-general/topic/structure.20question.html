I was just checking I'd got symmetry working with ne:

attribute [symm] ne.symm

/-
class zero_ne_one_class (α : Type u) extends has_zero α, has_one α :=
(zero_ne_one : 0 ≠ (1:α))
-/
example (R : Type) [zero_ne_one_class R] : (1 : R) ≠ 0 :=
begin
  symmetry,
  exact zero_ne_one,
end
