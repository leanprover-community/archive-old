Yury G. Kudryashov (Mar 29 2021 at 18:44):

Nobody needed it? def set.sigma {α : Type*} {β : α → Type*} (s : set α) (t : α → set β) := {x : Σ a, β a | x.1 ∈ s ∧ x.2 ∈ t x.1}
