Floris van Doorn (May 03 2021 at 07:05):

Yes, that is just some junk @[simps] generates and then records so that it doesn't have to generate it again.
It's not necessarily a problem if it is generated in a different file than where a structure is defined. Only when you declare custom projections or rename projections, you should do it in the same file.
