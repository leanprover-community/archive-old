here's a cool git feature I just found out about: if you do git pull --rebase --autostash (and you can set this as a global option), any uncommited local changes will get stashed, then the pull will happen, then you get the stash popped automatically. (if you set this as a git config, then leanproject up will do it for you too!) it also deals with git squashes messing up the history :) more details from somone gits better
