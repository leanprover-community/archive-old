Kevin Buzzard (Nov 02 2022 at 20:04):

Here is a way to get the simplifier to do it (which knows succ n = succ m <-> n = m):

example (m n : Nat) : Nat.succ n = Nat.succ m -> n = m := by
  intro h;
  simpa using h;

But the simplest way is not to do the intros:

example (m n : Nat) : Nat.succ n = Nat.succ m -> n = m := by simp
