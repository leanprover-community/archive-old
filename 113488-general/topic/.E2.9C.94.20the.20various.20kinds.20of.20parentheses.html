Jakub Kądziołka (Mar 11 2022 at 23:35):

I've seen parameters declared in Lean with four distinct kinds of parentheses: (), [], {}, and ⦃⦄. Are the exact differences between those documented anywhere? I tried searching #tpil for ⦃, but there weren't any results. So far, I think I figured out the following:

  • an argument declared with () needs to be provided explicitly
  • Lean will try to create a value for arguments declared with [] by using the typeclass mechanism
  • {} arguments are solved by unification
