Talking about a "set structure" doesn't have a precise meaning in Lean. But for all practical purpose these things are "sets" in the colloquial mathematical meaning of the word.
However, interior is only defined for terms of type set X, where X is topological space (strictly, a Type + instance of topological_space). Now this is maybe linguistically confusing, but set X means "set of terms of type X". In other words, set X is the "powerset" aka "subsets of X".
