Adam Topaz (Apr 23 2021 at 02:13):

Hi everyone,

I'm thinking about how to prove the first step of 8.4 from the blueprint aka 8.19 from analytic.pdf aka (essentially) the first step of 3.3 from condensed.pdf

The situation is this: we have a surjective map of finite sets f:X→Bf : X \to Bf:X→B. Obviously fff has a splitting, say σ:B→X\sigma : B \to Xσ:B→X.
Form the Cech nerve C(f)C(f)C(f). This is a simplicial object whose nnn-term is X×BX×B⋯×BXX \times_B X \times_B \cdots \times_B XX×B​X×B​⋯×B​X (n+1 times), and there is an augmentation C(f)→BC(f) \to BC(f)→B where BBB denotes the constant simplicial object with value BBB. We need to prove that the complex induced by C(f)C(f)C(f) (the alternating-face-map complex):

0→M^(B)→M^(X)→M^(X×BX)→⋯ 0 \to \hat M(B) \to \hat M(X) \to \hat M(X \times_B X) \to \cdots 0→M^(B)→M^(X)→M^(X×B​X)→⋯

is exact (in a strong sense, see below).

I'm fairly sure I see how to get the exactness using σ\sigmaσ explicitly, but it seems that one needs the stronger assertion that there is a contracting homotopy M^(C(f)n+1)→M^(C(f)n)\hat M(C(f)_{n+1}) \to \hat M(C(f)_n)M^(C(f)n+1​)→M^(C(f)n​) arising from applying M^\hat MM^ (a contravariant functor) to one map C(f)n→C(f)n+1C(f)_n \to C(f)_{n+1}C(f)n​→C(f)n+1​ (one for each nnn). The issue is that I'm failing to explicitly describe this single map C(f)n→C(f)n+1C(f)_n \to C(f)_{n+1}C(f)n​→C(f)n+1​ in terms of σ\sigmaσ. What am I missing?
