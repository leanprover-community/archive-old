Second question: Every term and type can be written as a string of letters and symbols. Some of these represent variables, and it just means that the only free variables that can occur in AiA_iAi​ must be in the set {x1,…,xi−1}\{x_1,\ldots,x_{i-1}\}{x1​,…,xi−1​}.
