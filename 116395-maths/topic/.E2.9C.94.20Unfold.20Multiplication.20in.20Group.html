Yakov Pechersky (Nov 01 2022 at 01:35):

The canonical way to help along with this is right after

instance : has_mul C := ⟨λ a b, ⟨a.val + b.val + a.val * b.val, sorry⟩⟩
-- you put
theorem mul_def (a b : C) (h) : a * b = ⟨a.val + b.val + a.val * b.val, h⟩ := rfl

and use that. You'll also likely need an extensionality lemma like:

theorem C.ext_iff (a b) (ha) (hb) : (⟨a, ha) : C) = (⟨b, hb) : C)  ↔ a = b := subtype.ext_iff
