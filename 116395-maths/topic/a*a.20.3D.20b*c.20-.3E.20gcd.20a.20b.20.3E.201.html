Kenny Lau (Aug 15 2020 at 17:19):

if gcd a b <= 1 then gcd a b = 0 or gcd a b = 1.

In the first case, we have a = 0 or b = 0, contradiction.

In the second case, we have gcd (a * a) b = 1, so gcd (b * c) b = 1, so b = 1, contradiction.
