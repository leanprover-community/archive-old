David Wärn said:

I guess what you want to know is that μ3(K)<∞\mu_3(K) < \inftyμ3​(K)<∞ for KKK compact?

This is easy: fix a nonempty open set UUU with compact closure KKK (it exists since the space is locally compact). Then μ3(U)≤μ1(K)\mu_3(U) \leq \mu_1(K)μ3​(U)≤μ1​(K) by monotonicity of μ1\mu_1μ1​, hence μ3(U)\mu_3(U)μ3​(U) is finite. Then, any compact set LLL is included in a finite number of left translates of UUU, and has therefore finite μ3\mu_3μ3​ measure.
