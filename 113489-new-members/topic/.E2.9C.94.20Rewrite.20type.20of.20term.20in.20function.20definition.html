Yes, that works! For anyone else stumbling on this thread following me along this dark path, the following is what you need:

def rewrite {α β : Type u} (heq : α = β) (a : α) : β
  := Eq.rec (motive := λ χ _ => χ) a heq

And in case you want to use some congruence, I have the following helper

def cong_rewrite {α : Type u} (a b : α) {f : α → Type u} (heq : a = b) (fa : f a) : f b
  := let hfeq : f a = f b := congrArg f heq
     rewrite hfeq ma
