Winston Yin (Jun 22 2021 at 08:58):

I managed to answer my own question...

lemma lem1 (x : M) (h : x ∈ p) : p := ⟨x, h⟩
def def2 (a : p) : M := by {cases a, exact a_val}
lemma lem2 (a : p) : def2 a ∈ p := by {cases a, exact a_property}
