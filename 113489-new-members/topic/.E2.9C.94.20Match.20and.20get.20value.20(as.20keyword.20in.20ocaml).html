Ryan Greenblatt (Dec 28 2021 at 02:47):

Is there something like the ocaml as keyword in lean?
For instance:

structure s := (fst : list ℕ) (snd : ℕ) (thd : ℕ)

-- better version
def compute : s → ℕ
| ({fst := fst::rst, ..}) as val := compute {fst := rst, ..val}
| ({thd := thd, ..}) := thd

-- sad version
def compute₂ : s → ℕ
| ({fst := fst::rst, snd := snd, thd := thd}) := compute₂ {fst := rst, snd := snd, thd := thd}
| ({thd := thd, ..}) := thd

Yury G. Kudryashov (Dec 28 2021 at 02:58):

Works:

structure s := (fst : list ℕ) (snd thd : ℕ)

def compute : s → ℕ
| (⟨[], _, a⟩) := a
| a@⟨_::t, b, c⟩ := compute { fst := t, .. a }
