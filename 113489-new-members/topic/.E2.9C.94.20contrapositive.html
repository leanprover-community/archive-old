it using what we've learnt so far is impossible (because it is not provable in
constructive logic).

My question is this, since I proved https://github.com/ImperialCollegeLondon/natural_number_game/blob/master/src/game/world6/level8.lean here lemma contrapositive (P Q : Prop) : (P → Q) → (¬ Q → ¬ P) can I not just apply contrapositive (¬ Q) (¬ P) to get (¬ Q → ¬ P) → (P → Q)?

Kevin Buzzard (Nov 27 2021 at 23:58):

no because then you'll get ¬ ¬ P -> ¬ ¬ Q and in constructive logic ¬ ¬ P isn't the same as P!
