I forgot about le_refl and tried to prove it using

#check (le_of_eq : y = z → y ≤ z)
so apparently I need to feed to it the proposition c=c but how do I do that? I tried using eq but failed.

Andrew Yang (Sep 27 2022 at 08:19):

eq is the type (statement); refl is the constructor (proof). So you should write refl c to obtain the proof of the statement c = c (eq c c). Alternatively you can also write rfl.
