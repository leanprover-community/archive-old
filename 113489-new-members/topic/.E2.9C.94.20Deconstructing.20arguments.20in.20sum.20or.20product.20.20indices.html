Stuart Presnell (Nov 16 2021 at 23:27):

I have a set S : finset (ℕ × ℕ) and a sum over S of the form ∑ x in S, ... (for some expression ... that uses the two components of the pair x). Since each x is an element of S it's of type ℕ × ℕ. So is it possible to write the sum in a way that automatically deconstructs the pair, rather than having to write x.1 and x.2 in the summand? I've tried a few things, like ∑ ⟨a,b⟩ in S, ..., but nothing has worked so far.

Kevin Buzzard (Nov 16 2021 at 23:33):

You can probably write something like let ⟨a,b⟩ := x in after the comma, but it's a bit meh

Yakov Pechersky (Nov 16 2021 at 23:33):

No, the sum notation does not currently support that, sorry

Eric Wieser (Nov 16 2021 at 23:34):

Using this type of destructuring often makes things annoying to prove
