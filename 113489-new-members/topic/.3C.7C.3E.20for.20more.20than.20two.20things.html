I don't know how to implement the highest function here:

-- A set of partial functions fᵢ.
def fᵢs : set (ℕ → option ℕ) := sorry

-- A priority for each fᵢ.
def priority (fᵢ : ℕ → option ℕ) : ℕ := sorry

-- Priorities are unique.
theorem unique_priority : ∀ (p : ℕ) (f f' : ℕ → option ℕ), (priority f = priority f') → f = f' := sorry

-- Returns the value `fᵢ(n)` for the fᵢ with the highest priority that doesn't map to `n` to `none`.
-- Or if all `fᵢ(n) = none`, then the value `none`.

If it were just two functions, I'd use <|>. And I feel like <|> could work here, too - but I don't know how.
