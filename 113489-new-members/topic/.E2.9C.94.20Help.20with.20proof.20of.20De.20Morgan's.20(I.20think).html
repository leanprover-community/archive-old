Marc Huisinga (Jan 05 2022 at 12:42):

Can you complete the proof from here?

variables (p q r : Prop)
example : ((p ∨ q) → r) → (p → r) ∧ (q → r) :=
λ h, _ -- we need to prove an implication, so we need to use assume / λ

example : ((p ∨ q) → r) → (p → r) ∧ (q → r) :=
λ h, and.intro _ _ -- we need to prove an `∧`, so we need to use `and.intro`

example : ((p ∨ q) → r) → (p → r) ∧ (q → r) :=
λ h, and.intro (λ hp, _) (λ hq, _) -- in both cases we need to prove an implication, so we need to use assume / λ

Marc Huisinga (Jan 05 2022 at 13:07):

or.elim is used when you want to use an ∨ to prove something else (i.e. case distinction), whereas or.inl and or.inr are used to prove ∨.
