Jason KY. (Sep 06 2020 at 18:32):

I would think you can start with something like 

import tactic

example (P: ℤ → Prop)
  (h8: ∀ (n : ℤ), P n → P (n + 8))
  (h3: ∀ (n : ℤ), P n → P (n - 3)) : (∀ (n : ℤ), P n) ∨ ∀ (n : ℤ), ¬P n :=
begin
  by_cases (P 0),
  left, sorry,
  right, sorry,
end
