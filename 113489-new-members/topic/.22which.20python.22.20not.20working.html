Mauricio Collares (Aug 29 2022 at 23:03):

You can try manually editing your PATH by going to the (old-style) Control Panel, choosing System, then Advanced, then Environment Variables. PATH should be one of the options, which you can then edit. If you have a C:\MyPython\bin\python.exe executable, then you should append C:\MyPython\bin to PATH as well as the correct separator. I don't remember if that's ; or : on Windows, but that should be easy to figure out from your current PATH variable.
