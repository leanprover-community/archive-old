Utensil Song (May 04 2020 at 14:28):

Hi, Jiekai, you may try:

git clone git@github.com:leanprover-community/tutorials.git
cd tutorials
mkdir -p _target/deps/mathlib
cd _target/deps/mathlib
git init
git remote add origin git@github.com:leanprover-community/mathlib.git
git fetch --depth 1 origin 2fc9b15898020df81bc1ec96b6820f5700956a62
git checkout FETCH_HEAD
cd ../../../
# Before executing the following, change the https url in `tutorials/leanpkg.toml` to `git@github.com:leanprover-community/mathlib.git`
leanpkg configure

The above will check out a shallow clone that contains no history but only the commit specified in tutorials/leanpkg.toml. This way saves most of the git clone traffic so it works under a peculiar internet connection.
