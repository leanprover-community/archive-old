def lift_option {α β : Type} (f : α → β) : option α → option β
|   none   := none
| (some a) := some (f a)

Chris B (Feb 28 2022 at 15:25):

This is functor.map.

Chris B (Feb 28 2022 at 15:25):

def lift_option {α β : Type} (f : α → β) : option α → option β := by library_search

also turns up option.map.
