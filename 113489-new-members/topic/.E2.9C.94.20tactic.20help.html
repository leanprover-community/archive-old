Kevin Buzzard (Jul 18 2021 at 17:18):

You are making a def with a match so this will clog up the actual code which defines isa. You can prove your example with unfold isa.

Alex J. Best (Jul 18 2021 at 17:19):

Any of the following proofs works

inductive AB : Type
| a : AB
| b : AB

def isa  : AB → Prop
| AB.a := true
| AB.b := false

example : isa AB.a :=
begin
  constructor,
end

example : isa AB.a :=
begin
  rw isa,
  trivial
end


example : isa AB.a :=
begin
  trivial
end
