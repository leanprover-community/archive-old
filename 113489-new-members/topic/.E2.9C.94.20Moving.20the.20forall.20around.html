Kyle Miller (Mar 07 2022 at 02:15):

If you really want a way to use imp_forall_iff, this seems to be a workaround:

...
  is_open_sUnion :=
  begin
    intros s ht j,
    have x := (f j).is_open_sUnion s,
    have ht' : ∀ t, (_ : Prop),
    { intro t,
      specialize ht t,
      rw imp_forall_iff at ht,
      exact ht, },
    rw forall_swap at ht',
    specialize ht' j,
    exact x ht',
  end
...
