Stuart Presnell (Dec 11 2021 at 16:02):

But if you're using by_cases, why not use by_cases p?

Solution

theorem t₈ : (p → (q ∨ r)) → ((p → q) ∨ (p → r)) :=
begin
intro hpqr,
by_cases hp : p,
{ cases hpqr hp,
{ left, intro, assumption },
{ right, intro, assumption } },
{ left, intro, contradiction },
end
