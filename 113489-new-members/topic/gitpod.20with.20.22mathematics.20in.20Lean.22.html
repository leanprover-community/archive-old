Mario Carneiro (Apr 05 2022 at 23:15):

Here's an example:

example (n : nat) : n < 1 + n :=
begin
  rw nat.add_comm,
end

If you put your cursor after the begin then the infoview should show the goal like this

1 goal
n: ℕ
⊢ n < 1 + n

and moving past the rw should change the 1 + n to n + 1
