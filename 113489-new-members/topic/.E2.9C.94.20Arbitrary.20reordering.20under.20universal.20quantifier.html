Patrick Johnson (Jan 31 2022 at 18:34):

Your second lemma test_ball_mem_comm' can be proved by tauto. I think the reason we have ball_mem_comm (and generally ball_cond_comm) is to handle quantifier conditions desugaring (since (∀ a b ∈ s, p a b) desugars to (∀ a, a ∈ s → ∀ b, b ∈ s → p a b)). In the "heterogeneous" case, you would write separate parameters anyway, so there's no point to have a conversion lemma.

Regarding your heterogeneous test1, you can prove it by showing that XS1 = f '' YS1 and XS2 = f '' YS2

lemma test1 (X Y : Type*) (XS1 XS2 : set X) (YS1 YS2 : set Y) (f : Y → X)
  (h1 : ∀ (P : X → Prop), (∀ x ∈ XS1, P x) ↔ (∀ y ∈ YS1, P (f y)))
  (h2 : ∀ (P : X → Prop), (∀ x ∈ XS2, P x) ↔ (∀ y ∈ YS2, P (f y)))
  (P : X → X → Prop) :
  (∀ x1 x2 (ha : x1 ∈ XS1) (hb : x2 ∈ XS2), P x1 x2) ↔
  (∀ y1 y2 (ha : y1 ∈ YS1) (hb : y2 ∈ YS2), P (f y1) (f y2)) :=
begin
  have h : ∀ {XS1 YS1} (h1 : ∀ (P : X → Prop),
  (∀ (x : X), x ∈ XS1 → P x) ↔ ∀ (y : Y), y ∈ YS1 → P (f y)), XS1 = f '' YS1,
  { rintro XS1 YS1 h1, ext x,
    use λ h, (h1 (λ x, x ∈ f '' YS1)).mpr (λ _ h, set.mem_image_of_mem f h) _ h,
    rintro ⟨y, h, rfl⟩, exact (h1 (λ x, x ∈ XS1)).mp (λ _, id) _ h },
  obtain rfl := h h1, obtain rfl := h h2,
  use λ h y1 y2 hy1 hy2, h _ _ ⟨_, hy1, rfl⟩ ⟨_, hy2, rfl⟩,
  rintro h x1 x2 ⟨y1, hy1, rfl⟩ ⟨y2, hy2, rfl⟩, exact h _ _ hy1 hy2,
end

Patrick Johnson (Jan 31 2022 at 19:51):

You can make another definition and prove that it is equivalent to the original definition:

import probability_theory.independence
open probability_theory

def indep_sets' {α : Type*} [measurable_space α]
  (s₁ s₂ : set (set α)) (μ : measure_theory.measure α) :=
∀ (t₁ ∈ s₁) (t₂ ∈ s₂), μ (t₁ ∩ t₂) = (μ t₁) * μ t₂

example : @indep_sets = @indep_sets' :=
by ext; split; exact λ a b c d, a b d c
