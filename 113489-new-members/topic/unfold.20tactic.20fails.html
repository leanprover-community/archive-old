Hello! This is my first post in Zulip chat. I'm just wondering why does unfold f not work in this example?

def f : ℕ → ℕ → Prop
| 0 0 := true
| (n + 1) (m + 1) := f n m
| _ _ := false

lemma lem1 : ¬f 0 1 := begin
  -- unfold f,
  exact id,
end
