Anne Baanen (Sep 16 2021 at 09:12):

erd1 said:

But why does it work at https://leanprover.github.io/live/latest/ ?

More specifically, leanprover.github.io is for the official version of Lean run by Microsoft Research, the Mathlib community runs leanprover-community.github.io. While the official team is working on Lean 4, the community has been updating Lean 3. So the page you went to is the latest official version 3.4.1, which is very far from the latest community version 3.33.
