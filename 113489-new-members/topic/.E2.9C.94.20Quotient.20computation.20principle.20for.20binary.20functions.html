Henrik Böving (Apr 17 2022 at 12:56):

You can just simp further:

  theorem add.comm: ∀x y: MyInt, x + y = y + x := by
    intro x y
    simp_all [HAdd.hAdd, add]
    induction x using Quot.ind with | mk x =>
    induction y using Quot.ind with | mk y =>
    simp [Quotient.liftOn₂, Quotient.lift₂, Quotient.lift]
    sorry

and it will reach:

case mk.mk
x y : RawInt
⊢ MyInt.addRawInts x y = MyInt.addRawInts y x

though I'm not entirely sure whether that is the intended way
