theorem subtract_to_equation: ∀{a b c: Nat}, a + c = b + c → a = b := by
  intro a b c acbc
  induction c with
  | zero => simp_all
  | succ pc inner =>
    rw [Nat.add_comm a (Nat.succ pc), Nat.add_comm b (Nat.succ pc)] at acbc
    simp [Nat.succ_add] at acbc
    simp [Nat.add_comm] at acbc
    exact (inner acbc)

@Arthur Paulino I managed to bash something together:

theorem subtract_to_equation: ∀{a b c: Nat}, a + c = b + c → a = b := by
  intro a b c acbc
  cases h : c with
  | zero => cases h; exact acbc
  | succ pc =>
    have : pc < c := by cases h; apply Nat.lt_succ_self
    have ih : a + pc = b + pc → a = b := subtract_to_equation
    cases h
    injection acbc with h
    exact ih h
termination_by _ => c

Kyle Miller (Apr 12 2022 at 21:15):

@Arthur Paulino I guess what @Eric Wieser said applies somewhat to Lean 4. If you use pattern matching for c you can simplify this proof:

theorem subtract_to_equation: ∀{a b c: Nat}, a + c = b + c → a = b
| a, b, 0, acbc => acbc
| a, b, (pc + 1), acbc => by
  have ih : a + pc = b + pc → a = b := subtract_to_equation
  simp [← Nat.add_assoc] at acbc
  simp [← Nat.succ_eq_add_one] at acbc
  exact ih acbc
