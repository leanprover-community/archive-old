Kyle Miller (Oct 31 2022 at 02:25):

These seem to work for me:

example : set ℕ := {(2 * n) | (n : ℕ)}

def f (n : ℕ) : ℕ := 2 * n
example : set ℕ := {(f n) | (n : ℕ)}

example (s : set ℕ) := {(2 * n) | (n ∈ s)}
