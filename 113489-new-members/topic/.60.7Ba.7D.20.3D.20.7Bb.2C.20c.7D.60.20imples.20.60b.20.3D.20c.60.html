prewarning, also a new-member. I would break it up into smaller parts.
we have a_2 \in {a_1} since we can rw it. Likewise for b_2. So they should both be equal to a_1
I put the haves in and then used library_search to find the name of set.mem_singleton_iff.mp

lemma helper (h : ({a₁} : set α) = {a₂, b₂}) : a₂ = b₂ := begin
  have a2incl : a₂ ∈ {a₁},
  {
    rw h,
    simp,
  },
  have a2eq : a₂ = a₁,
  {
    exact set.mem_singleton_iff.mp a2incl,
  },
  rw a2eq,
  have b2incl : b₂ ∈ {a₁},
  {
    rw h,
    simp,
  },
  have b2eq : b₂ = a₁,
  {
    exact set.mem_singleton_iff.mp b2incl,
  },
  exact b2eq.symm,
end

Here's one:

lemma helper (h : ({a₁} : set α) = {a₂, b₂}) : a₂ = b₂ :=
begin
  have h' := set.ext_iff.mp h a₂,
  have h'' := set.ext_iff.mp h b₂,
  tidy,

@Sigma Here's a minimized version of your proof:

lemma helper (h : ({a₁} : set α) = {a₂, b₂}) : a₂ = b₂ := begin
  obtain rfl := set.mem_singleton_iff.mp (by { rw h, simp } : a₂ ∈ {a₁}),
  obtain rfl := set.mem_singleton_iff.mp (by { rw h, simp } : b₂ ∈ {a₂}),
  refl,
end

(obtain rfl := is a way to immediately substitute everywhere when the result is an equality)

Patrick Johnson (Jan 01 2022 at 01:44):

@Pedro Minicz Note that {a₁} is definitionally equal to (λx, x = a₁) and {a₂, b₂} is definitionally equal to (λx, x = a₂ ∨ x = b₂), so you can also prove it using docs#congr_fun:

lemma helper (h : ({a₁} : set α) = {a₂, b₂}) : a₂ = b₂ :=
by change (λx, x = a₁) = (λx, x = a₂ ∨ x = b₂) at h;
finish using [(⟨congr_fun h a₂, congr_fun h b₂⟩ : _ ∧ _)]

lemma helper (h : ({a₁} : set α) = {a₂, b₂}) : a₂ = b₂ :=
