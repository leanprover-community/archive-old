Mario Carneiro (Sep 28 2020 at 22:10):

You can use `[simp] to create a tactic unit object which calls the interactive simp tactic

Mario Carneiro (Sep 28 2020 at 22:10):

i.e.

def my_tac : tactic unit := do
  x <- bla,
  `[simp],
  exact x

Scott Morrison (Oct 02 2020 at 03:45):
