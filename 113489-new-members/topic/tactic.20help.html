Yakov Pechersky (Jul 19 2021 at 02:24):

unfold doesn't see through semireducible definitions. So you have to do:

example : is_same_term (term.ite term.t term.t term.f) v10 :=
begin
  rw v10,
  unfold is_same_term,
  refine ⟨trivial, _, _⟩,
  { rw v00,
    rw is_same_term,
    trivial },
  { rw v01,
    rw is_same_term,
    trivial }
end
