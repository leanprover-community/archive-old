This used to work in the past (there was a nat.pow in core Lean), but it was really annoying how nat.pow was not definitionally equal to monoid.pow, so after the fork in 2019 we removed it from core.

When I'm doing mathematics I start every file with import tactic and encourage my students to do the same, precisely so they don't keep asking me questions like this. import tactic is a catch-all import which gives you a lot of stuff which you would expect to be there if you're doing mathematics.
